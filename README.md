# WAMOCON GmbH (Static Export)

This folder contains the exported static pages for the WAMOCON website.

## Online reference (original)

- https://wamocon.com

## Deploy

Use GitHub Pages (or any static hosting) and set the site root to this folder.
`index.html` is provided as an English “pages index” so you can open every exported `page*.html`.

Note: internal navigation links inside the exported pages may use “pretty” routes (e.g. `/azubifae`).
If you need those exact routes on static hosting, you may need server-side rewrites/redirects.
